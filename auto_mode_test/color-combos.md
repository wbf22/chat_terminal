Color Combos — Ready-to-use Palettes

A curated set of great color combinations with hex codes, usage notes, and small CSS snippets. Use these as starting points for UI, branding, or layout work. Each palette includes suggestions for background, text, primary, secondary, and accent colors plus brief accessibility notes.

1) Classic Neutrals
- Background: #FFFFFF (white)
- Text: #111111 (almost black)
- Primary: #2E2E2E (charcoal)
- Secondary: #7D7D7D (mid gray)
- Accent: #C49A6C (warm tan)
Usage: Clean UIs, corporate sites, documentation.
Contrast notes: High contrast between text and background for readability.
CSS:
:root { --bg: #FFFFFF; --text: #111111; --primary: #2E2E2E; --secondary: #7D7D7D; --accent: #C49A6C; }

Notes and tips
- Always test color contrast for accessibility (WCAG AA/AAA) especially for body text and form controls.
- Use the accent color sparingly for actions, highlights, or icons.
- Consider color-blind safe variants if your audience requires it (avoid relying solely on red/green distinctions).
